nw.js application demonstrating CMake.js out-of-the-box nw.js compatibility

This nw.js application uses CMake.js demo module (cmake-js-tut-01-module) from the first tutorial. There is no nw-gyp like magic required, all CMake.js native modules are compatible with nw.js out-of-the-box. See the third CMake.js tutorial for details.
